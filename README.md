# Финальный Проект: Комплексный Анализ Данных в Academica Courses
Описание Проекта
Данный репозиторий содержит финальный проект по курсу "Аналитик данных" от Academica Courses. 
Проект демонстрирует полный цикл работы с данными, начиная от извлечения и трансформации, до анализа, визуализации и подготовки отчетов. 
Основная цель — выявить ключевые бизнес-инсайты, связанные с поведением клиентов и эффективностью продаж.

Стек Технологий
Базы данных: MySQL
Инструменты анализа: Microsoft Excel, Google Sheets
Языки Программирования: Python (Pandas, NumPy, Matplotlib)
Инструменты Визуализации: Power BI
Метрики: LTV, Retention Rate, ARPU, DAU/MAU, A/B тесты.

## Выполненные Задачи и Основные Блоки Проекта

### 1. Блок SQL (Файл: `SQL FP - DINA.sql`)
Цель: Анализ клиентской базы и транзакций для выявления паттернов поведения.
Задачи:
    * Вывод списка клиентов с непрерывной годовой историей транзакций.
    * Расчет среднего чека, средней суммы покупок за месяц и общего количества операций по клиенту.
    * Анализ транзакций в разрезе месяцев: средний чек, среднее количество операций, количество уникальных клиентов.
    * Определение долей от общего количества операций и сумм транзакций.
    * Анализ гендерного и возрастного состава клиентов с их долями затрат (M/F/NA, возрастные группы с шагом 10 лет) поквартально и за весь период.

### 2. Блок Excel (Файлы: `Блок Excel FP - DINA.xlsx`, `Блок Excel FP - DINA`)
Цель: Консолидация и агрегация данных для операционной отчетности.
Задачи:
    * Совмещение данных из нескольких листов.
    * Формирование сводных таблиц в разрезе недель и территорий.
    * Выявление ТОП-3 территорий по доле в товарообороте и по товарообороту на склад за последнюю неделю.
    * Построение графиков понедельной динамики товарооборота.

### 3. Блок Power BI (Файл: `POWER BI FP - DINA DONE.pbix`)
Цель: Создание интерактивной панели для мониторинга ключевых бизнес-показателей.
Задачи:
    * Разработка интерактивных дашбордов для отслеживания динамики продаж и клиентского поведения.
    * Внедрение более 15 кастомных мер на языке DAX для детального анализа.
    * Обеспечение принятия решений на основе актуальных данных в режиме Real-Time.

### 4. Блок Python (Файл: `PYTHON FP - DINA.ipynb`)
Цель: Глубокий разведочный анализ данных и их предобработка.
Задачи:
    * Проведение разведочного анализа данных.
    * Очистка датасетов, устранение более 15% аномалий и дублей.
    * Использование библиотек Pandas, NumPy, Matplotlib для манипуляций с данными и их визуализации.
    * Посмотреть отчет в PDF - https://github.com/dinauzivert/FINAL-PROJECTS---DINA-NURGALIYEVA/blob/main/POWER%20BI%20FP%20-%20DINA%20.pdf

### 5. Блок Тестовое задание (Файл: `БЛОК ТЗ - DINA.docx`)
Цель: Демонстрация аналитического мышления и умения отвечать на бизнес-вопросы.
Содержание: Ответы на вопросы тестового задания, представленного в формате Word.

## Результаты и Выводы
* Выявлены ключевые сегменты клиентов с разной историей покупок.
* Определены территориальные лидеры и динамика их товарооборота.
* Разработаны инструменты для оперативного мониторинга бизнес-метрик.
* Проанализированы и представлены выводы по продуктовым метрикам и A/B тестам.
